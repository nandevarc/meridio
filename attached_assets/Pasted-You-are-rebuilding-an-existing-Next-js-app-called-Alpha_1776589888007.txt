GENERAL RULES:

1. All forms: no HTML <form> tags, use div + onClick handlers only
2. All inputs: bg --bg-input, border --border, rounded-xl, padding 10px 12px, 
   text --text-primary, 13px, IBM Plex Mono. Focus: border --border-focus.
   No default browser outline (outline: none).
3. All select elements: custom styled, same as inputs, no default arrow.
   Add a custom chevron icon.
4. Tap targets: minimum 44x44px for all interactive elements
5. No page reloads on any action — all state updates happen in React state 
   and persist to localStorage
6. Do not use any animation library. CSS transitions only.
   Transitions: all interactive elements have transition: all 150ms ease
7. Scrollbar: hide on all overflow-x-auto containers. Show thin (2px) 
   red scrollbar on overflow-y containers.
8. -webkit-tap-highlight-color: transparent on all interactive elements
9. Font import: add to global CSS, not inline
10. ID generation: use crypto.randomUUID() or fallback to 
    Date.now().toString(36) + Math.random().toString(36).substr(2)
11. All dates displayed in format: "DD MMM YYYY" (e.g. "19 Apr 2026")
12. All datetimes displayed in format: "DD/MM/YYYY, HH.MM" 
    (e.g. "19/04/2026, 15.56")
